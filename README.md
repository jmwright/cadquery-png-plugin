# cadquery-png-plugin
PNG export plugin for CadQuery

## Running Tests

1. Clone the repository.
```
https://github.com/jmwright/cadquery-png-plugin.git && cd cadquery-png-plugin
```

2. Install the plugin and its development dependencies.
```
pip install -e .
pip install -e .[dev]
```

3. Run the tests
```
pytest
```
